## 初期対応範囲

- ROS 2 Jazzy Jalisco
- Ubuntu 24.04 LTS
- Python `rclpy` / C++ `rclcpp`
- colcon、rosdep、rqt、rviz2、rosbag2
- URDF、TF2、ros2_control、MoveIt 2
- Gazebo、NVIDIA Isaac Sim
- 産業用ロボットとSim-to-Real
