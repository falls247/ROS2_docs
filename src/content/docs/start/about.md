## 目的

公式ドキュメントの単純な翻訳ではなく、ROS 2初心者が「なぜ必要か」「内部で何が起きるか」「どう確認するか」を理解できる実践教材を目指す。

## 対象読者

- ROS 2を初めて学ぶソフトウェア・制御技術者
- 産業用ロボットとROS 2の接続を検討する技術者
- MoveIt 2、Gazebo、Isaac Simを使う開発者
- 実機導入前にシミュレーションで検証したい担当者

## 基準環境

| 項目 | 基準 |
|---|---|
| OS | Ubuntu 24.04 LTS |
| ROS 2 | Jazzy Jalisco |
| Python | ROS 2 Jazzy標準環境 |
| ビルド | colcon |
| パッケージ管理 | apt / rosdep |

## 記事の品質基準

各実践ページは次を含む。

1. 概念と利用目的
2. 前提環境
3. 実行手順
4. 期待される結果
5. 内部で起きていること
6. よくあるエラー
7. 次に読むページ

各ページには最終検証日を記録し、手順や対象環境を再確認した際に更新する。
